The SCM project is open-source and free for all amateur swimming clubs.

This is a worthy community project. It will not only aids the clubs that use it, but also support the sport of swimming.

SwimClubMeet

This installer contains the core application.

It also includes 'Help for SCM'.

SCM_Marshal

Rally your swimmers. Disqualify or scratch entrants, pool side. A Windows 64bit application designed for MS Edge Pro touch pad.

SCM_TimeKeeper

Record your swimmers racetimes, pool side. A Windows 64bit application designed for MS Edge Pro touch pad.

SCM_LeaderBoard

Display in real time, members place and score, house scores and event details. Option to group by division.

SCM_BuildMeAClub

Creates the SwimClubMeet database on your MS SQLEXPRESS server.

SCM_Update

A utility to version up your SwimClubMeet database on MS SQLEXPRESS.
