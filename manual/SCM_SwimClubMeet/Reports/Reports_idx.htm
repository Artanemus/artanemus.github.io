A very long list of examples for you to view ....
Sessions 2xVersions ... When club night is over, you print one of these reports and send to you club's committee
Events 2xVersions ... Version 1 is a summary of events. Version 2 is detailed.
Heats 2xVersions ... Version 1 is a detailed report for the current heat. Version 2 lists all the heats in the current event. They both show racetime, if they have been entered.
Marshal 2xVersions ... These reports can be used to rally your members for the start of the event. Version 2 is also suitable for your Chief Timekeeper to use.
Timekeeper 2xVersions ... These reports can be used to enter racetimes and to mark if an entrant was scratched or disqualified. Typically used by your Chief Timekeeper or Referee.
Nominate 2xVersions This is a list of nominations for the session.

Note: A nomination isn't an indication that the member was given an lane in the event.
