Tools - House

Create, delete houses. Edit house names, motto, color, logo, etc. Assign members to houses. Print house reports.

What the point of houses in SCM?

The answer is none. Swimming Clubs don't need houses to run club-nights. Using houses would complicate the meet, add more work and have little gain.

Houses Details - Tab-Sheet

Houses are a work-in-progress and have been placed in the release version to encourage a little exploration and experimentation for its users.

Houses are used by the SCM_LeaderBoard application. Very useful if you were a school trying to run a swimming carnival. Not that SCM is designed or intended to be used for this purpose.ðŸ˜‰

On first install, SCM has placed some starter data into the house table. Consider this data as examples. You'll discover four houses have been created. You can navigate these houses using the navigation bar placed at the bottom of the dialogue.

Picture format for the logo is JPEG, BMP, SVG. Recommended size is 800x800 pixels square.

Click the house color swatch to open the color picker.

The house members table lists all of the club members who have been assigned to this house. This table is read only. To enter a member to a house, you must use the Members Dialogue. Here you'll find a house dropdown combo-box in the Members Details Tab-Sheet and in the Members List Tab-Sheet.

Use the search box to find a member - should the list be very long.

The Navigation bar enables delete and insert of new houses.

Future versions will including a special tool to 'Auto-Assign' members to a house. This will be based on a weighted algorithm that will create an even spread of age, gender, and performance.

To see a House leader-board score use the SCM_LeaderBoard application. House scores can also be printed in this app.

House Reports Tab-Sheet
