Session Buttons

This section is closely tied to the Session Grid topic. For more details on each button's actions, view the topic ... link.

A short summary of the session buttons.

Toggle Visibility.

When you hover over this button a hint message will appear.Toggle the visibility of closed sessions.. The button will change appearance when sessions are made invisible.

Lock Session.

When you hover over this button a hint message will appear. Lock-Unlock the session. (CTRL+L). Locked session are colored tomato red.

Create New Session.

When you hover over this button a hint message will appear. Create a new session. (Ctrl-Ins). When clicked, the New Session Dialogue will open.

Delete Session.

When you hover over this button a hint message will appear. Delete the current selected session. (Ctrl+Del). A confirmation dialogue appears, asking you if you want to do the delete.

This button will be grayed (disabled) if the session has been locked.

View Session Report.

When you hover over this button a hint message will appear. Display the session report.. Using CNTRL+LMOUSE click will display the alternative version of the session report.
