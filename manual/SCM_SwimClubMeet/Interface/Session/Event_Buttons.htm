Event Buttons

This section is closely tied to the Event Grid topic. For more details on each button's actions, view the topic ... link.

A short summary of the event buttons.

Expand the grid.

When you hover over this button a hint message will appear.Toggle expansion of the event grid.. When expanded you have the ability to enter event descriptions.

Move Up.

When you hover over this button a hint message will appear. Move the event up. (Ctrl+Up).

This button will be grayed (disabled) if the session has been locked.

Move down.

When you hover over this button a hint message will appear. Move the event down. (Ctrl+Down).

This button will be grayed (disabled) if the session has been locked.

Create New Event.

When you hover over this button a hint message will appear. Create a new event. (Ctrl-Ins).

New events are placed at the end of the grid. They are auto-numbered once you finalize the edit.

This button will be grayed (disabled) if the session has been locked.

Delete Event.

When you hover over this button a hint message will appear. Delete the selected event. (Ctrl+Del). A confirmation dialogue appears, asking you if you want to do the delete.

This button will be grayed (disabled) if the session has been locked.

View Event Report.

When you hover over this button a hint message will appear. Display the event report.. Using CNTRL+LMOUSE click will display the alternative version of the event report.
