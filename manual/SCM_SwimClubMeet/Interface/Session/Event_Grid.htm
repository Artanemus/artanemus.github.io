The Event Grid.

Create, enter distance and stroke, order, delete and print reports.

LEDGER DESCRIPTIONS ....
1. Event Number Read only. Event number. Auto-assigned.
2. Distance Distance in metres to swim. Enter by keyboard or select from the drop-down combobox.
3. Stroke Swimming stroke. Enter by keyboard or select from the drop-down combobox.
4. Event Status Read only. When checked, it indicates that all heats are finalized (closed).
5. Nominees, Entrants. Both cell are read only.

Nominees - The number of members who nominated to for the event.

Entrants - The number of nominees who were given a lane. ie. the number of entrants.

6. Event buttons. Expand grid, move up, move down, create, delete, print event report.

Event grid ... pop-up menu.

Hover over the grid then RMouse click to access the context sensitive pop-up menu.

Create new event

To create a new event
  • Click the New Event button
  • Hover over the event grid, then RMOUSE click and select New Event from the popup menu.
  • Or move to the event line and arrow down
  • Or move to the last field on the last event and press tab.

As new events are inserted, they are auto-matically assigned an event number.

Delete an event.

To Delete an Event...
  • Select the event to delete. Click the Delete Event button
  • Hover over the event grid, then RMOUSE click and select Delete Event from the popup menu.
  • Select the event to delete. then press CTRL-DELETE.

In all these methods, a dialogue will be shown to confirm the deletion. WARNING: There is no undo.

If the heats contained in the event are closed or raced , you cannot delete the event.

If the session is locked, then you can't delete events.

Note: When an event is deleted, entrants, racetimes, nomination data and heats for that event is removed.

Expand the events table.

Press this button to toggle the expansion of the event's table. When expanded, additional event fields are available.

Note: on future versions of SCM, the event type field will be available. In this field you can select between individual and team events.

Grid Column - Event Description.

This column is only visible when the events grid is expanded.

Descriptions placed in this column are displayed in dialogues, reports, and in the support application Nominate, Marshall, TimeKeeper, etc. They are displayed everywhere.

Why the description column?

Normally, event descriptions are not needed. But if you are running identical events, then this column is required. For example, you have 2x50M Freestyle events. One for boys under 14 years and one for girls under 14 years. You'll need to describe these events, else caous.

SCM can be told to place an auto description, each time you create a new event. The text used can be found in the Tools âž¡ Options âž¡ System âž¡ Event_Grid. You can modify this text.

Note: on future versions of SCM, you'll be able to use meta-data to customize the descriptions based on the event type. (wit. auto-descriptions will be a little more useful.)

When SCM creates finals, semi-finals and quarter-finals, it will write an auto-description. SCM adds these descriptions so the special events are delineated from the seed event.

Grid column - Distance.

SCM's default distances are 25m, 50m, 100m, 200m, 400m, 1000m. Distances are not arbitrary, they can be changed or added too in the SCM MSSQL database using SSMS.

How to edit distance.

  • Using mouse. LMOUSE click the column. LMOUSE click the drop-down combobox widget. Then LMOUSE click the distance in the stack.
  • Using keyboard. Navigate to the column using ARROW keys or TAB. Press F2 to enter edit mode. Select the distance using the short-cut keys 0, 1, 2, 4, 5 keys. Finally press ENTER. Note: there is no short-cut key for 1000m.

Grid Column - Stroke

SCM's default strokes are Freestyle, Breaststroke, Backstroke and Butterfly. Swimming strokes are not arbitrary, they can be changed or added too in the SCM MSSQL database using SSMS.

How to editing stroke.

  • Using mouse. LMOUSE click the column. LMOUSE click the drop-down combobox widget. Then LMOUSE click the stroke in the stack.
  • Using keyboard. Navigate to the column using ARROW keys or TAB. Press F2 to enter edit mode. Select the distance using the short-cut keys f, r, a, u, m. Finally press ENTER.
