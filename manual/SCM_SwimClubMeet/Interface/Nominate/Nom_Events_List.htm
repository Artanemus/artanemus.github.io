Nominate - Events List

When checked and painted tomatoe red, the club member has been nominated for the event.

DESCRIPTION FOR LEDGER ....
1. Nominee The name and age of the current selected member seeking nomination.
2. CheckBox Toggle the check box to nominate the member for the event.
3. Qualified Status The letter 'Q' indicates the member is 'qualified' for this event.
4. Event Distance, stroke and event description (if given).

Nominee Events List

This list displays all the events that have been assigned to the current selected session. The sort order is identical to the events grid seen on session tab-sheet. Displayed is the distance, stroke, and event description (if given).

You may add, delete and move events at any time during the meet. This list will always be up to date.

Notice that each time you select a different member, the list's contents is updated.

Above the event list is the name of the current selected member and age, as of the current session date. As you browse members this status text gets updated.

Qualification Status (Q).

If the Qualified button is enabled and the member has swum the qualifiy distance within the qualify time - then the letter Q appears.

To learn more about how qualification works and how to set it up, follow this link.
