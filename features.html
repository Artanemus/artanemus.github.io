Simple, Clean UI

Parents and friends don't need to be IT experts to use SCM.

All Pools

Short or long course. Any length, any lane count. With the option to ignore gutter lanes.

All Session Data Stored

Sessions, events, nominations, heats, entrants and race times are all recorded.

Full Reporting

All data can be printed or exported to PDF. Over 20 reports to chose from.

Member's Data

All swimming history and basic details for each members is recorded.

Auto-Build Heats

It couldn't be easier. Single click to create heats. Or build heats manually.

Swimmer Qualifications

Automatically shows the qualification of members as they nominate.

SQL Open-Source

Your MS SQLEXPRESS SwimClubMeet database is not encrypted. You have full ownership of your data.

Run Paperless

Use the support applications - Marshall and Timekeeper and never print paper again.

Removes the Nomination Bottleneck

SCM scales. Add more laptops/touch-pads. Use the Self-Nomination app.

Teamed Events

Build relay events. Create teams and select the swimmers. Auto-Build for teamed events coming soon.

Split Timing

Lap times not implemented. The database is complete but no UI. Ready for future implementation.

FINA Codes

Use simplified disqualification method or enter FINA codes (made easy with descriptions and context aware).

Timing Pads

Currently no connectivity to electronic timing pads.
