About the developer

I'm an old guy who's been writing code for way to long. My first coding experience was writing machine code on MS assembler. The code was written for a Z80 that run on a 100 bus computer - which I had built.

My main skill is building business applications. Often as the sole developer. I dive into the client's knowledge base and extract the business model. Which I twist and wrangle into a database. The last step is the construction of the interface. Oh, and lots of reports.

C++ and Delphi are my preferred programming languages, though I have used many others. If you included scripting, like SQL, powershell, etc, then number increases.
