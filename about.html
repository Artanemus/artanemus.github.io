About the developer

It's very simple. I'm an old guy who's been writing code for way to long. My first coding experience was writing machine code on MS assembler. The code was written for a Z80 that run on a 100 bus computer - which I had built. Yes, I'm really old.

My main skill is to build business applications. Often as the sole developer. I dive into the client's knowledge base and extract the business model. Which I twist and wangle into a database. The last step is the construction of the interface. Oh, and lots of reports.

I seldom write phone apps and rarely write API's. I'm always in demand.

I write good code.
